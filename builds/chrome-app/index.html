EasyRPG Player is a RPG Maker 2000/2003 interpreter. Other RPG Maker versions, like XP, VX Ace and MV are not supported.

Open the settings tab and add the folder that contains your RPG Maker 2000/2003 games. Ensure that all your games extracted. The formats EXE, ZIP and RAR are not supported. An extracted game contains the file "RPG_RT.ldb" in its main directory.

If you have the Runtime Package (RTP) you can add the one for 2000 or 2003 in the settings tab. The RTP is a shared collection of assets that are used by some games.
